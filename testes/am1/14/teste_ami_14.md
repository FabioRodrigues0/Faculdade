> **Disciplina:** Análise Matemática I (AMI)

**1.** Determine o domínio de cada uma das funções, justificando a sua resposta.

a) $f(x) = \dfrac{1}{\sqrt{x^2 - 9}}$

> [!note]- Espaço para resposta

b) $g(x) = \ln(2x - 3) + \sqrt{x + 1}$

> [!note]- Espaço para resposta

**2.** Calcule os seguintes limites:

a) $\displaystyle\lim_{x \to 3} \dfrac{x^2 - 9}{x^2 - 5x + 6}$

> [!note]- Espaço para resposta

b) $\displaystyle\lim_{x \to +\infty} \dfrac{3x^3 - 2x + 1}{6x^3 + x^2 - 4}$

> [!note]- Espaço para resposta

c) $\displaystyle\lim_{x \to 0} \dfrac{\sqrt{1 + x} - 1}{x}$

> [!note]- Espaço para resposta

---

**3.** Seja $f : \mathbb{R} \to \mathbb{R}$ definida por:

$$f(x) = \begin{cases} \dfrac{x^2 - 4}{x - 2}, & \text{se } x \neq 2 \\ k, & \text{se } x = 2 \end{cases}$$

Determine o valor de $k$ para que $f$ seja contínua em $x = 2$. Justifique a sua resposta.

> [!note]- Espaço para resposta

---

**4.** Seja $h$ uma função de domínio $\mathbb{R}$ definida por $h(x) = x^3 - 3x + 1$.

Utilizando o Teorema de Bolzano, mostre que a função possui pelo menos um zero no intervalo $]0, 1[$.

> [!note]- Espaço para resposta

**5.** Considere a função $f(x) = \dfrac{2x^2 + x - 1}{x + 2}$.
