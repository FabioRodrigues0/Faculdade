# Teste 11 — Análise Matemática I

> [!info] Informação
> **Disciplina:** Análise Matemática I

**2.** Calcule os seguintes limites:

a) $\displaystyle \lim_{x \to 2} \frac{x^2 - 4}{x^2 - 3x + 2}$

> [!note]- Espaço para resposta

b) $\displaystyle \lim_{x \to +\infty} \frac{4x^2 + 3x - 1}{2x^2 - x + 5}$

> [!note]- Espaço para resposta

c) $\displaystyle \lim_{x \to 0^+} \frac{x^2 - 1}{x}$

> [!note]- Espaço para resposta

**3.** Seja $f : \mathbb{R} \to \mathbb{R}$ definida por:

$$f(x) = \begin{cases} \dfrac{x^2 - x - 6}{x - 3} & \text{se } x \neq 3 \\ k & \text{se } x = 3 \end{cases}$$

Determine o valor de $k$ para que $f$ seja contínua em $x = 3$. Justifique.

> [!note]- Espaço para resposta

**4.** Seja $h$ uma função de domínio $\mathbb{R}$ definida por $h(x) = x^3 + 2x - 5$.

Utilizando o Teorema de Bolzano, mostre que a função possui pelo menos um zero no intervalo $]1, 2[$.

> [!note]- Espaço para resposta
